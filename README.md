# B站字幕下载助手

一个简单好用的 Chrome 扩展，在 B 站视频页面**一键下载字幕**，可导出为 **SRT（带时间轴）** 或 **TXT（纯文本）**，并支持多 P / 合集视频逐集下载。

---

## ✨ 功能特点

- **一键下载**：在视频页面点一下，直接拿到字幕文件
- **两种格式**：
  - `SRT`：带序号和时间轴，可直接给播放器 / 剪辑软件用
  - `TXT`：纯文字逐行排列，适合整理文稿、喂给 AI 总结
- **多 P / 合集支持**：开启「多 P 模式」可选择任意分集单独下载
- **自动处理签名**：内置 B 站 WBI 签名算法，无需手动抓包

---

## 📦 安装方法

本扩展未上架商店，需以「开发者模式」加载：

1. 下载本项目（`Code → Download ZIP`）并解压，或 `git clone` 到本地
2. 打开 Chrome，地址栏输入 `chrome://extensions/` 回车
3. 打开右上角的 **「开发者模式」** 开关
4. 点击 **「加载已解压的扩展程序」**
5. 选择本项目文件夹（含 `manifest.json` 的那一层）
6. 扩展栏出现「B站字幕下载助手」图标即安装成功（可点击拼图图标固定到工具栏）

> 💡 修改代码后，回到 `chrome://extensions/` 点该扩展的「刷新」按钮即可生效。

## 🚀 使用方法

1. **登录 B 站**：先在浏览器里登录账号（字幕接口需要登录态）
2. **打开视频**：进入任意 `bilibili.com/video/...` 视频页面
3. **点击扩展图标**，在弹窗中按需设置：
   | 选项 | 说明 |
   |------|------|
   | **启用字幕下载** | 打开此开关后，下方「下载字幕」按钮才可点击 |
   | **多 P 模式** | 关闭＝下载当前/首个分 P；开启＝弹出分集列表，逐集选择下载 |
   | **导出格式** | 选择 `SRT（带时间轴）` 或 `TXT（纯文本）` |
4. **点击「下载字幕」**：
   - 单 P 模式：自动下载，文件出现在浏览器下载目录
   - 多 P 模式：在视频页面弹出分集列表，点对应分集的「下载字幕」即可
5. 下载的文件名形如 `BVxxxxxxxxxx_分集标题.srt`（或 `.txt`）

## 📄 两种格式对比

**SRT** —— 带序号与时间轴，适合播放器加载、视频剪辑：
```
1
00:00:01,200 --> 00:00:03,500
大家好，欢迎来到本期视频

2
00:00:03,500 --> 00:00:06,000
今天我们来聊聊……
```

**TXT** —— 仅文字逐行排列，适合阅读、整理、AI 总结：
```
大家好，欢迎来到本期视频
今天我们来聊聊……
```

## ❓ 常见问题

- **点了没反应 / 提示「助手未加载」**：刷新一下视频页面再试（扩展需要注入页面后才能工作）。
- **提示「未找到字幕」**：该视频本身没有字幕（包括没有 AI 字幕），无法下载。
- **下载的不是想要的分 P**：单 P 模式只下首个分集，请开启「多 P 模式」后手动选择。
- **接口报错 / 拿不到字幕**：确认已登录 B 站；如仍失败，刷新页面让扩展重新捕获签名参数后再试。

## 🛠 技术说明

- 基于 **Manifest V3**，内容脚本注入 `*.bilibili.com` 视频页
- 通过 `x/web-interface/nav` 获取并重排 `mixin_key`，对 `x/player/wbi/v2` 请求做 **WBI 签名（MD5）**
- 后台 `background.js` 监听播放器请求，自动捕获动态指纹参数以提高成功率
- 字幕 JSON 在本地转换为 SRT / TXT 后由浏览器下载，全程不经过第三方服务器

## 📁 文件结构

```
├── manifest.json                 # 扩展配置（MV3）
├── popup.html / popup.css / popup.js   # 弹窗界面与交互
├── content.js                    # 注入视频页的核心脚本（下载逻辑）
├── bilibili-subtitle-helper.js   # WBI 签名与字幕处理核心类
├── background.js                 # 后台脚本，捕获动态签名参数
└── icons/                        # 扩展图标
```
